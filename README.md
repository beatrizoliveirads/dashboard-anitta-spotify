# Dashboard Anitta — Spotify Analytics | Power BI

Dashboard interativo de análise de performance da Anitta no Spotify, desenvolvido no Power BI com dados coletados manualmente de fontes públicas.

## Objetivo
Analisar a presença global da Anitta no Spotify, identificando músicas mais ouvidas, países com maior alcance e evolução de streams ao longo dos anos.

## Indicadores (KPIs)
- Total de Streams — 4 bilhões
- Ouvintes Mensais — 40 milhões
- Seguidores — 14 milhões
- Músicas com +100 milhões de streams — 45

## Visuais do Dashboard
- Top músicas mais ouvidas
- Ouvintes por país — mapa global
- Evolução de streams por ano
- Distribuição de streams por país

## Ferramentas utilizadas
- Power BI Desktop
- DAX
- Power Query
- Excel (coleta e organização dos dados)

## Fonte dos dados
Dados coletados manualmente de fontes públicas: Chartmasters, Spotify e pesquisas abertas.
